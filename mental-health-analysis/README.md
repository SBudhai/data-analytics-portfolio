# Mental Health in the Workplace Analysis

## Overview

This project analyzes mental health survey data to understand how workplace and demographic factors influence treatment-seeking behavior.

## Tools Used

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Jupyter Notebook

## Key Insights

* Many individuals report seeking mental health treatment
* Workplace interference is strongly associated with treatment behavior
* Access to employer benefits may influence treatment decisions
